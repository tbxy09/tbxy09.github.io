==================

##__format__ 格式
>级别化，格式化内容

   * bold  <kbd>Clt</kbd>+<kbd>shift</kbd>+<kbd>B</kbd>
      `
      __format__

      
![]()

   * Header <kbd>Clt</kbd>+<kbd>Num</kbd>
      
      `#一级标题
       ##二级标题
       ###三级标题
       ####四级标题`

   * italics <kbd>clt</kbd>+<kbd>shift</kbd>+<kbd>I</kbd>
       
       `
       _italic_
       `
       
   * Image Ref  <kbd>super</kbd>+<kbd>shift</kbd>+<kbd>K</kbd>
        `
        ![]()
        `
   * Single line shading Shading 打阴影<kbd>shift</kbd>+<kbd><</kbd>
        `
        >shading
        `
   * 单个tab回退 <kbd>shift</kbd>+<kbd>tab</kbd>
   * code引用 <kbd>```</kbd>+<kbd>```</kbd>
   * singline code 引用 <kbd>`</kbd>+<kbd>`</kbd>
   * footnote<kbd>alt</kbd>+<kbd>shift</kbd>+<kbd>6</kbd>
   * link ref <kbd>win</kbd>+<kbd>clt</kbd>+<kbd>v</kbd>
   * git preview <kbd>clt</kbd>+<kbd>shift</kbd>+<kbd>G</kbd>
